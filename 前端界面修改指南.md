# 前端界面修改指南

## 📁 前端文件位置

所有前端文件都在 `app/templates/` 目录下：

```
app/templates/
├── base.html              # 基础模板（包含导航栏、样式等）
├── index.html             # 首页
├── accounts.html          # 科目列表页面
├── transactions.html      # 交易列表页面
├── transaction_new.html   # 新增交易页面
├── reports_balances.html  # 科目余额报表
└── reports_transaction_details.html  # 交易明细报表
```

## 🎨 如何修改样式（颜色、字体、布局）

### 1. 修改全局样式

打开 `app/templates/base.html` 文件，找到 `<style>` 标签（大约在第 9 行）。

**可以修改的内容：**

```css
:root {
    --primary: #2563eb;        /* 主色调（蓝色）*/
    --primary-dark: #1d4ed8;   /* 深色主色调 */
    --bg-light: #f7f7f7;       /* 背景浅色 */
    --bg-card: #ffffff;        /* 卡片背景色 */
    --border-color: #e2e8f0;   /* 边框颜色 */
    --text-muted: #64748b;     /* 次要文字颜色 */
}
```

**示例：改成绿色主题**
```css
:root {
    --primary: #10b981;        /* 改成绿色 */
    --primary-dark: #059669;
    /* ... 其他颜色 */
}
```

### 2. 修改导航栏样式

在 `base.html` 的 `<style>` 中，找到 `header` 和 `nav` 相关样式：

```css
header {
    background: var(--primary);  /* 导航栏背景色 */
    color: #fff;                /* 文字颜色 */
    padding: 16px 24px;         /* 内边距 */
}

nav a {
    color: rgba(255, 255, 255, 0.95);  /* 链接颜色 */
    padding: 6px 12px;                  /* 链接内边距 */
    font-size: 14px;                    /* 字体大小 */
}
```

### 3. 修改表格样式

在 `base.html` 的 `<style>` 中，找到 `table` 相关样式：

```css
table {
    width: 100%;
    border-collapse: collapse;
    margin-top: 16px;
}

th {
    background: #f1f5f9;  /* 表头背景色 */
    text-align: left;
}
```

## 📄 如何修改页面内容

### 1. 修改页面标题

在每个页面的开头，找到：
```html
{% block title %}科目余额报表 - 简易财务记账系统{% endblock %}
```
修改引号内的文字即可。

### 2. 修改页面标题和描述

在报表页面中，找到：
```html
<h2 class="mb-4">📊 科目余额报表</h2>
<p class="text-muted">显示所有科目的当前余额情况。</p>
```
修改这些文字即可。

### 3. 修改表格列名

在报表页面中，找到表格的 `<thead>` 部分：
```html
<thead class="table-primary">
    <tr>
        <th scope="col">科目名称</th>
        <th scope="col">科目类别</th>
        <th scope="col">余额</th>
    </tr>
</thead>
```
修改 `<th>` 标签内的文字即可。

### 4. 修改导航栏链接

在 `base.html` 中，找到 `<nav>` 部分（大约在第 147 行）：
```html
<nav>
    <a href="/">首页</a>
    <a href="/accounts/view">科目列表</a>
    <a href="/transactions/view">交易列表</a>
    <!-- ... 更多链接 -->
</nav>
```
可以：
- 修改链接文字
- 添加新链接
- 删除不需要的链接

## 🛠️ 常见修改示例

### 示例 1：改变主题颜色为绿色

在 `base.html` 的 `<style>` 中修改：
```css
:root {
    --primary: #10b981;      /* 改成绿色 */
    --primary-dark: #059669;
}
```

### 示例 2：修改导航栏高度

在 `base.html` 的 `<style>` 中修改：
```css
header {
    padding: 20px 24px;  /* 增加内边距，导航栏会更高 */
}
```

### 示例 3：修改表格字体大小

在 `base.html` 的 `<style>` 中修改：
```css
th, td {
    font-size: 16px;  /* 改成更大的字体 */
}
```

### 示例 4：添加自定义样式到报表页面

在报表页面的 `{% block content %}` 开头添加：
```html
{% block content %}
<style>
    /* 只在这个页面生效的样式 */
    .my-custom-class {
        color: red;
        font-weight: bold;
    }
</style>
<!-- 页面内容 -->
```

### 示例 5：修改按钮样式

在 `base.html` 的 `<style>` 中，找到 `.btn` 样式：
```css
.btn {
    background: var(--primary);
    color: #fff;
    padding: 8px 16px;
    border-radius: 6px;
}
```

## 💡 修改技巧

### 1. 实时预览
修改后保存文件，如果应用正在运行（使用 `--reload` 模式），页面会自动刷新。

### 2. 使用浏览器开发者工具
1. 在浏览器中按 `F12` 打开开发者工具
2. 使用"检查元素"功能查看样式
3. 在开发者工具中临时修改样式，找到满意的效果
4. 将修改后的 CSS 复制到模板文件中

### 3. 使用 Bootstrap 类
项目使用了 Bootstrap 5，可以直接使用 Bootstrap 的 CSS 类：
- `text-center` - 文字居中
- `text-primary` - 主色调文字
- `bg-light` - 浅色背景
- `shadow` - 阴影效果
- `rounded` - 圆角
- 等等...

## 📝 修改步骤总结

1. **确定要修改的文件**
   - 全局样式 → `base.html`
   - 特定页面 → 对应的 `.html` 文件

2. **找到要修改的位置**
   - 样式 → `<style>` 标签内
   - 内容 → HTML 标签内

3. **进行修改**
   - 修改颜色值、文字、样式等

4. **保存文件**
   - 保存后刷新浏览器查看效果

5. **调试**
   - 如果效果不对，使用浏览器开发者工具检查

## ⚠️ 注意事项

1. **备份文件**：修改前建议先备份原文件
2. **语法正确**：确保 HTML 和 CSS 语法正确
3. **测试**：修改后在不同浏览器中测试效果
4. **不要删除**：不要删除 `{% block %}` 和 `{% extends %}` 等 Jinja2 模板语法
